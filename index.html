Цельсий > Фаренгейт

Задать температуру в градусах по Цельсию. Вывести в alert соответствующую температуру в градусах по Фаренгейту.

Василий админ?

Объявить две переменные: admin и name. Записать в name строку «Василий». Скопировать значение из name в admin. Вывести в консоль переменную admin.

Почему так выходит?

Вывести в консоль значения выражений и объяснить почему получились такие значения используя комментарии к каждому выражению.

Именование переменных

Напишите, являются ли данные имена переменных корректными:

Согласно правилам, идентификатор переменной должен начинаться с буквы (A-Z, a-z), нижнего подчёркивания (_) или знака доллара ($) последующими символами могу быть в том числе и цифры.

Переменной mode присваивается строка со значением «normal» let mode = "normal";

Переменной my_valu3 присваивается число со значением 102 let my_valu3 = 102;

Переменная некорректна. Переменной 3my_value3 ничего не присвоится, т.к. нарушено правило let 3my_value3 = "102";

Переменной __hello__ присваивается строка со значением «world» let __hello__ = "world";

Переменной $$$ присваивается строка со значением «money» let $$$ = "money";

Переменная некорректна.!0_world ничего не присвоится, т.к. нарушено правило let !0_world = true;
